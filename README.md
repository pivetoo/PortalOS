# PortalOS

Sistema de gerenciamento de ordens de servico, clientes e projetos.

## Estrutura do Projeto

```
PortalOS/
├── PortalOS.API/           # ASP.NET Core API
├── PortalOS.Domain/        # Entidades, Services, ViewModels
├── PortalOS.Infrastructure/# Migrations, Mappings
└── PortalOS.Web/           # Frontend React + TypeScript
```

## Tecnologias

**Backend:**
- .NET 8.0
- NHibernate ORM
- PostgreSQL
- dNET Framework

**Frontend:**
- React 19
- TypeScript
- Vite
- dRTS (biblioteca de componentes)
